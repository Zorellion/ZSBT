# Pets

Pets controls how pet/guardian damage is displayed.

## Where to configure
- `/zsbt` -> `Pets`

## Options
- **Enabled**: turn pet scrolling text on/off.
- **Scroll Area**: where pet messages appear (often `Outgoing`).
- **Aggregation**: how pet hits are grouped (for example, a generic summary).
- **Min Threshold**: hide small pet hits.
- **Merge Window**: if available, controls how tightly hits are combined.
- **Show Count**: show a hit count on merged messages.

## Tips
- If pet spam is overwhelming, raise `Min Threshold` and use a larger merge window.
